        <p style="font-style: italic; color: #64748b; margin: 0 0 0.6em 0;">First-author project · AI for Science · 2026 · in progress</p>
        <p>
            An end-to-end multi-agent pipeline for automated powder X-ray diffraction (PXRD) analysis on real measured spectra. Xtalyst integrates phase identification, Rietveld refinement, and conformal property prediction, diagnosing and bridging the simulation-to-real gap to lift crystal-system accuracy from 27.5% to 49.4%. Validated on a frozen held-out set of 534 measured spectra.
        </p>
        <ul>
            <li>Multi-agent architecture with tool-augmented reasoning for scientific data analysis.</li>
            <li>Manuscript in preparation for <strong>Science Advances</strong>. Project website and demo coming soon.</li>
        </ul>
